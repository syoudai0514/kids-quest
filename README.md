# ほしぞらクエスト 🚀⭐🦕

5歳（年長）〜小1向けの「毎日ミッション学習アプリ」。宇宙と恐竜を旅しながら、
相棒モンスター **ホッシュ** といっしょに「よむ・かく・すうじ」を学びます。
「勉強」ではなく「あそびの延長」に感じられることを最優先に設計しています。

> Android タブレットのブラウザで動く **React + PWA（オフライン対応）**。
> アカウント登録不要・データは端末内のみに保存。
> 公開URL: https://syoudai0514.github.io/kids-quest/

---

## あそびの流れ

1. **はじめて**: 相棒ホッシュと出会い、すきな色を選んで出発（オンボーディング）
2. **ホーム＝宇宙の旅のハブ**: いまいる惑星で空の色が変わる。
   きょうのミッション5つ＝**航路マップ**をロケットが進み、次の惑星に到着すると
   物語が進んで、なかまが増える（全10惑星）
3. **タスク**: 1つ4問・1〜2分。正解のたびに褒め＋✨ほしのかけら（XP）
4. **相棒が育つ**: XPでレベルアップ＋進化。連続日数🔥ストリークも表示
5. **息抜きバトル**: 1日3回＋「ついか問題」で得たチケット分。
   勝つと「ほしのわ」でモンスターを捕まえ、**ずかん100体**を埋めていく

## 「頭が良くなる」ための仕掛け

- **アダプティブ難易度**: 2連続正解でレベルアップ（得意はどんどん先へ）。
  ミスしても責めず、ヒント→正解を光らせ**音声で解説**（苦手意識を作らない）
- **復習キュー（想起練習）**: 間違えた問題を記録し、後のタスクで確率的に再出題。
  できたらキューから消える
- **算数教具の視覚化**: かぞえる・たしざんは**5個ずつの列**（5のかたまり感覚）、
  10づくりは**10のフレーム**（2×5マス）で見せる
- **書字の3段階**: **お手本アニメ（文字が浮かび上がる＋書きはじめの点が光る）→
  なぞり → 自由書き**。星1〜3の評価は必ずもらえる
- **バトルの三すくみ**: 🔥→🌿→💧→🔥（⭐は安定）。相手の属性を見て
  効くわざを推論する＝遊びながら論理を学ぶ

## いまのコンテンツ量（小1レベルまで）

- **よむ**: 単語82語（絵と言葉が必ず1対1）＋小1漢字の読み48字
- **かく**: ひらがな・カタカナ全部＋小1漢字40字のなぞり書き
- **すうじ**: かぞえる／くらべっこ（絵・100までの数字）／たしざん／
  **くり上がり・くり下がり**（〜20）／10づくり／数のならび
- **モンスター**: 100体（すべてSVGの独自デザイン。既存IP不使用）

## うごかし方

```bash
npm install
npm run dev        # 開発サーバ
npm run build && npm run preview   # 本番ビルド確認
```

`main` に push すると GitHub Actions が自動で GitHub Pages にデプロイします。
タブレットでは Chrome の「ホーム画面に追加」で全画面・オフラインのアプリになります。

## 設計（拡張しやすさ重視）

```
src/
├─ engine/
│  ├─ tts.js          日本語の音声読み上げ（指示・問題・正誤は必ず声でも）
│  ├─ sfx.js          効果音＋モンスターの鳴き声（Web Audio合成・ファイル不要）
│  ├─ storage.js      localStorage 永続化＋日付ロールオーバー
│  ├─ difficulty.js   難易度の自動調整（分野ごとの習熟度）
│  ├─ battle.js       バトルの属性相性・ダメージ計算
│  ├─ activities.js   学習分野レジストリ ← 分野を足すならここ
│  └─ missions.js     毎日ミッションの組み立て（コア5＋おかわり上限6）
├─ data/
│  ├─ monsters.js     モンスター100体（手描き8＋決定論生成92）
│  ├─ planets.js      惑星10個（空の色・物語・なかま解放）← 足すだけで増える
│  └─ content/
│     ├─ reading.js   よむ: 単語82＋漢字48＋復習キュー対応ジェネレータ
│     ├─ numbers.js   すうじ: 9種の出題タイプ（レベルで解禁）
│     └─ writing.js   かく: 文字プール＋復習キュー対応
├─ components/
│  ├─ Monster.jsx         モンスターSVG（絵柄8種）
│  ├─ QuestionVisual.jsx  問題の共通描画（グリッド/10フレーム/漢字…）
│  ├─ TracingCanvas.jsx   お手本→なぞり→自由書きキャンバス
│  └─ common.jsx          星背景・紙吹雪・進捗ドット
├─ screens/
│  ├─ OnboardingScreen.jsx  はじめての おはなし
│  ├─ HomeScreen.jsx        宇宙の旅ハブ（航路マップ・相棒・Lv）
│  ├─ ActivityPlayer.jsx    出題エンジン（復習・解説・コンボ）
│  ├─ BattleScreen.jsx      息抜きバトル（三すくみ・捕獲）
│  ├─ CollectionScreen.jsx  ずかん（鳴き声つき）
│  ├─ ParentScreen.jsx      保護者ビュー
│  └─ CelebrationOverlay.jsx ごほうび演出
└─ state/
   └─ GameContext.jsx  状態の中枢（XP/ストリーク/復習キュー/セーブ移行）
```

### 拡張のしかた

- **単語/漢字/文字/惑星/モンスターを足す** → 各 data ファイルの配列に1件追加するだけ
  - よむの単語は「絵文字と言葉が1対1」「同義語なし」「絵文字重複なし」を守ること
- **新しい出題タイプ** → `numbers.js` の kinds に追加し、`QuestionVisual.jsx` に
  描画が必要なら kind を1つ足す
- **新しい分野** → `data/content/` にジェネレータを作り `engine/activities.js` に登録
- コンテンツを大きく変えたら `GameContext.jsx` の `CONTENT_VERSION` を上げる
  （進捗・ずかん・レベルは保持したまま、当日ミッションだけ作り直される）

## 著作権について

モンスター・世界観・バトルはすべて独自デザインです。
既存IP（ポケモン等）の画像・名称・データは一切使用していません。
